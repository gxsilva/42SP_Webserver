##  CLANG FORMAT
Is a code formating tool that automatically reformats your _C, C++, Java,JavaScript, Objective-C_ and ither languages according to a set of style rules

it's part of the _LLVM Project_ (The same that includes the Clang compiler), the goal is to make the code
- consistent
- readable
- standardized

```bash
clang-format -i main.cpp

#Or just

clang-format main.cpp
```

`-i` = "in-place"


LLVM -> The default fallback style for clang-format | somenting standard minimal
